# 圖片與 GIF

這頁說明如何插入圖片和 GIF。

## 圖片放置位置

建議所有圖片都放在：

```text
docs/images/
```

例如：

```text
docs/images/demo.png
docs/images/demo.gif
```

## 在文章中插入圖片

如果文章在：

```text
docs/tutorials/images-and-gifs.md
```

圖片在：

```text
docs/images/demo.png
```

寫法是：

```md
![圖片說明](../images/demo.png)
```

## 插入 GIF

GIF 寫法跟圖片一樣：

```md
![操作示範](../images/demo.gif)
```

## 建議

- 圖片檔名建議用英文或數字
- 避免空白，例如不要用 `我的 圖片.png`
- 可以用 `step-01.png`、`login-demo.gif` 這種命名
