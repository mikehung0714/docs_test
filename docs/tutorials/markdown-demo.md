# Markdown 範例

這頁示範常用 Markdown 寫法。

## 標題

```md
# 一級標題
## 二級標題
### 三級標題
```

## 清單

```md
- 第一項
- 第二項
- 第三項
```

顯示效果：

- 第一項
- 第二項
- 第三項

## 編號清單

```md
1. 第一步
2. 第二步
3. 第三步
```

顯示效果：

1. 第一步
2. 第二步
3. 第三步

## 表格

```md
| 功能 | 支援 |
|---|---|
| Markdown | 是 |
| 圖片 | 是 |
| GIF | 是 |
```

顯示效果：

| 功能 | 支援 |
|---|---|
| Markdown | 是 |
| 圖片 | 是 |
| GIF | 是 |

## 提示區塊

VitePress 支援這種提示區塊。

::: tip 小提示
這是補充說明。
:::

::: warning 注意
這是需要注意的地方。
:::

::: danger 危險
這是高風險操作提醒。
:::
